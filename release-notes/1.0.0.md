# App2Docker 1.0.0

**发布日期**：（发布时填写）

## 本版说明

- 统一应用版本来源：`backend/VERSION`
- 登录后主界面**标题旁**与**页脚**展示当前版本号，支持打开「版本与更新」查看 Gitee 最新 Release 与发行正文
- 登录页展示当前版本（公开接口 `GET /api/public/version`）
- 发现新版本时页内提示（Toast，同一会话同一边远端版本仅提示一次）

## 升级说明

### Docker

1. 备份宿主机映射的数据目录（如 `./data` → 容器内 `/app/data`）
2. 拉取新镜像并重建容器，保持卷与端口配置不变

### 源码 / 本地

1. 拉取对应标签代码
2. 按仓库根目录 `README.md` 安装依赖、构建前端并启动后端

## 破坏性变更

**无**（首版基线）。

发版时请以 [`backend/VERSION`](../backend/VERSION) 为准更新版本号，并在 Gitee 创建对应 Release。
