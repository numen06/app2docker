# App2Docker 1.0.2

**发布日期**：（发布时填写）

## 新功能与改进

- **镜像迁移**：源/目标仓库间同步镜像，支持路径、标签、Cron 定时与任务复制
- **流水线复制**：一键复制流水线配置（新 Webhook；构建后回调需重新配置）
- **团队**：邀请链接续期与过期处理优化；团队设置、所有者与删除流程完善
- **导出**：导出流程与体验优化
- **界面**：管理后台与关键页面移动端布局优化

## 版本机制

- 应用版本号与 Gitee Release 对齐为 **1.0.2**（以 [`backend/VERSION`](../backend/VERSION) 为准）
- 标题区/页脚/登录页展示版本；「版本与更新」可查看 Gitee Release 与 `release-notes` 说明

## 从 1.0.0 / 1.0.1 升级

1. 备份 `data` 等持久化目录
2. Docker：拉取 `1.0.2` 对应镜像（或 `latest`）后重建容器，卷映射不变
3. 源码：拉取 `v1.0.2` 标签，重新构建前端并重启后端

## 破坏性变更

**无**（与 1.0.0 数据与配置兼容）。

发版时请同步在 Gitee 创建 **v1.0.2** Release，并将本文件要点复制到发行说明。
