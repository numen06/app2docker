## 本版亮点（2.x）

- **镜像迁移**：源/目标仓库间同步镜像，支持路径、标签、Cron 定时与任务复制
- **流水线复制**：一键复制流水线配置（新 Webhook；构建后回调需重新配置）
- **团队与权限**：邀请链接续期与过期处理；团队设置、所有者转让与删除流程完善
- **体验**：管理后台与关键页面移动端适配；对外首页与 README 同步新能力说明

## 修复与优化

- 导出流程与体验优化
- 构建提交与团队相关若干问题修复

## 版本机制

- 应用版本号与 Gitee Release 对齐为 **2.0.1**（以 [`backend/VERSION`](../backend/VERSION) 为准）
- 标题区/页脚/登录页展示版本；「版本与更新」可查看 Gitee Release 与 `release-notes` 说明

## 从 1.0.x 升级

1. 备份 `data` 等持久化目录
2. Docker：拉取 `2.0.1` 对应镜像（或 `latest`）后重建容器，卷映射不变
3. 源码：拉取 `v2.0.1` 标签，重新构建前端并重启后端

## 破坏性变更

**无**（与 1.0.0 数据与配置兼容）。
